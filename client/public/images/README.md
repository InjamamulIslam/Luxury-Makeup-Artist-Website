You may also want to add:

## Image Specifications

- **Format**: JPG or WebP recommended
- **Size**: Optimize images for web (recommend 1920px wide for hero, 800px for others)
- **Quality**: 80-85% JPEG quality is ideal for web

## How to Add Images

1. Save your images with the exact filenames listed above
2. Place them in: `client/public/images/`
3. The website will automatically display them

All image paths in the website code have been updated to use these filenames.
